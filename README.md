<h1 align="center">Social Media Database Project</h1>

# About❓
What I'm building: I am designing and building the foundational database architecture for a modern social media platform.

What I'm managing: My schema actively manages complex data networks across multiple users. This includes tracking followers, profile interests, and real-time public activities like posts, post-likes, comments, comment-likes, hashtag follows, and bookmarks.

How it connects: Using structured SQL queries, I have mapped out the clear connections and relationships between different user behaviors and activities on the platform.

Real-world handling: To mimic real-world production environments, I have structured the database to store multimedia data in its true storage format (such as routing images and videos via hosted URLs).

My Goal: Through focused research and design, I have created a clean, precise, and optimized blueprint of a basic social networking system.
